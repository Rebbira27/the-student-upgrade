# The Student Upgrade

A free educational platform for students — habit tracking, study planner, English stories, and a resource library with exam sheets.

## Built By
Rebira — Grade 11 Student, Ethiopia

## What This Platform Includes

- **Upgrade Log** — Daily reflection with feeling tracking (1-5), weekly charts, schedule parser
- **Study Planner** — Spaced repetition system (Day 1, 3, 7, 14, 30)
- **Read & Upgrade** — Ethiopian folk tales and student-submitted stories with reflection questions
- **Resource Library** — Exam sheets, notes, worksheets, organized by grade and subject
- **Admin Dashboard** — User stats, resource management, story review system

## Tech Stack

- HTML, CSS, JavaScript
- PHP
- MySQL
- Gmail SMTP for password reset
